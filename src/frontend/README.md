# frontend

## 项目环境配置

执行如下命令配置前端运行环境：

```
npm install
```

`npm` 会自动完成依赖配置，之后使用命令：

```
npm run serve
```

运行前端服务，并在浏览器中查看。

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
